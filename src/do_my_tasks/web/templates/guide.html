Guide

DMT CLI 사용법 및 웹 대시보드 안내

1 빠른 시작

dmt web 웹 대시보드 실행 (기본 포트 7317)
dmt collect 오늘의 세션 & 커밋 데이터 수집
dmt sessions watch 실시간 세션 모니터링 (idle 감지 + 알림)
dmt sessions watch --web watch + 웹 대시보드 동시 실행

⬇ dmt collect

Claude 세션 로그와 Git 커밋을 DB에 수집합니다. 대시보드에 데이터를 표시하려면 이 명령을 먼저 실행하세요.

-d, --date YYYY-MM-DD 특정 날짜 수집 (기본값: 오늘)
-p, --project NAME 특정 프로젝트만 수집

⌨ dmt sessions

sessions live 현재 실행 중인 Claude 세션 표시
-w, --wide CWD 등 상세 정보 표시
-d, --detail 로그 상태(tool, 파일 등) 포함 표시
sessions watch 세션 감시 + idle 시 알림
-i, --interval SEC 갱신 주기 (기본값: 10초)
--idle SEC idle 판단 기준 시간 (기본값: 30초)
-t, --tail 상단 고정 패널 + 하단 스크롤 로그
-w, --web 웹 대시보드 동시 실행
--web-port PORT 웹 포트 지정 (기본값: 7317)
sessions clean idle 세션 종료
-i, --idle MIN idle 기준 시간 (기본값: 60분)
-n, --dry-run 실제 종료 없이 대상만 출력
-f, --force 확인 없이 강제 종료

✓ dmt tasks

tasks add "제목" -p PROJECT -P high|medium|low -d 설명
tasks list -p PROJECT -s STATUS -d DATE
tasks complete TASK_ID 태스크 완료 처리 (T-0001 또는 숫자)
tasks update TASK_ID -s STATUS -P PRIORITY
tasks delete TASK_ID 태스크 삭제 (-f 확인 생략)
tasks rollover 미완료 태스크 다음날로 이월

⚙ dmt config

config add PATH 프로젝트 등록 (-n NAME -b BRANCH)
config list 등록된 프로젝트 목록
config remove NAME 프로젝트 등록 해제
config discover ~/.claude/projects/에서 프로젝트 자동 탐색
config show 현재 설정 확인

◉ 세션 상태

🔄 working Claude가 응답을 생성 중입니다
⏸ idle 작업 완료, 사용자 입력 대기 중입니다
⚠️ waiting 툴 실행 승인 대기 중입니다
💤 inactive 최근 활동이 없는 세션입니다

🗄 데이터 저장 위치

데이터베이스
~/.config/do_my_tasks/data.db

SQLite — 세션, 커밋, 태스크, 프로젝트 정보 저장

환경 변수
DMT_DB_PATH

DB 경로를 커스텀하려면 이 환경 변수를 설정하세요

Claude 로그
~/.claude/projects/

Claude Code가 기록하는 세션 JSONL 로그 파일
